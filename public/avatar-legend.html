Manual agents

Update An Agent From A Script

Add an agent above, copy its token, then call this endpoint whenever the agent changes state. The office view reads the saved state and refreshes it with the other agents.

URL
/api/agent-state
States
Working, Blocked, Sleeping, Reading, Gaming, Coffee break, Listening, Walking
curl -X POST http://localhost:3000/api/agent-state \
  -H 'Content-Type: application/json' \
  -d '{"token":"AGENT_TOKEN","state":"Working","task":"Handling queue item 42"}'

Use the token shown on the manual agent row. The optional task field becomes the short status text in the office.
